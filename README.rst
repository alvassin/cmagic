Installation
------------

Ubuntu/Debian
+++++++++++++

.. code-block:: bash

   apt-get install -y libmagic-dev


Centos
++++++

.. code-block:: bash

   yum install -y file-devel


Macos
+++++

.. code-block:: bash

   brew install libmagic
